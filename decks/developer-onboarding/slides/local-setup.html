03 · Workflow
Local Setup & Environment
Four steps from a clean machine to a running application — clone, configure, install, start.
01
Clone the repository
Pull the primary application repo (and any supporting services).
$ git clone
   [Repository URL]
$ cd [Repo Folder]
$ git checkout -b
   feature/[ticket]
02
Configure environment
Copy the env template and fill values from the secrets manager.
$ cp .env.example
   .env
# set DB, API,
# SECRET_KEY values
# never commit .env
03
Install dependencies
Install backend, frontend, and Python service packages.
$ ./gradlew build
$ npm install
$ python -m venv .venv
$ pip install -r
   requirements.txt
04
Start the application
Run backend, services, and frontend in the recommended order.
$ ./gradlew bootRun
$ python app.py
$ npm start
→ http://localhost
   :3000 / :8080
