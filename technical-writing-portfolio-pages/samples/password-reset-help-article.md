# How to Reset Your Password and Re-enable Multi-Factor Authentication

## Summary

If you cannot sign in because you forgot your password or lost access to your authenticator app, you can reset your password and set up multi-factor authentication again in a few minutes.

## Reset Your Password

1. Go to the sign-in page.
2. Select **Forgot Password**.
3. Enter the email address associated with your account.
4. Open the password reset email and select the reset link.
5. Create a new password that meets the listed security requirements.

## Re-enable Multi-Factor Authentication

After resetting your password:

1. Sign in with your new password.
2. Go to **Account Settings**.
3. Select **Security**.
4. Choose **Set Up Multi-Factor Authentication**.
5. Scan the QR code with your authenticator app.
6. Enter the verification code to complete setup.

## Troubleshooting

If you do not receive the password reset email:

- check your spam or junk folder
- confirm that you entered the correct email address
- wait a few minutes and try again

If your verification code does not work:

- make sure your mobile device time is set automatically
- generate a new code and try again
- contact support if the issue continues

## Result

Once setup is complete, your account will require both your password and a time-based verification code at sign-in.
