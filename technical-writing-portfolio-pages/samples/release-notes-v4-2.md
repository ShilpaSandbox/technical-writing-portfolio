## Release Date

May 8, 2026

## Overview

Version 4.2 introduces dashboard filtering improvements, faster report generation, and several usability fixes across account settings and notifications.

## New Features

### Saved dashboard filters

Users can now save custom dashboard filters and reuse them without rebuilding the same view each time.

### Export history panel

The Reports page now includes an export history panel so users can quickly re-download recently generated files.

## Improvements

### Faster monthly reports

Monthly summary reports now generate more quickly for large datasets.

### Improved notification settings layout

Notification preferences are now grouped by category to make configuration easier.

## Bug Fixes

- Fixed an issue that caused duplicate email notifications in some workflows
- Fixed a display issue affecting date pickers in Safari
- Fixed an account settings bug where profile updates did not immediately appear after save

## Known Issues

CSV exports with more than 100,000 rows may take longer than expected during peak traffic periods.
